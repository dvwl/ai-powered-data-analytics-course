## Getting Started

### Prerequisites
- Python 3.7+ installed
- Jupyter Notebook or JupyterLab
- Required packages: `pandas`, `numpy`, `matplotlib`, `seaborn`, `plotly`

### Installation
```bash
pip install pandas numpy matplotlib seaborn plotly jupyter
```
